# Clario

Рабочий MVP веб-приложения на Vite + React + TypeScript.

## Запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```

## Что умеет MVP

- принимает исходное описание задачи;
- задает вопросы по одному на экран;
- поддерживает готовые варианты, свой вариант, свободный ввод, "не знаю" и пропуск;
- собирает структурированное ТЗ;
- копирует ТЗ в буфер обмена;
- скачивает ТЗ в `.txt`;
- начинает бриф заново без перезагрузки страницы.
